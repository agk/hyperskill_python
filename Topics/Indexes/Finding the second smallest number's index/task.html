Given a list of integers, find the index of the second smallest number in the list. If the second smallest number does not exist, return -1. You can assume that the list size is less than or equal to 10^3 and each integer in the list is less than or equal to 10^3. The list will be given as a string with each number separated by a space.
Sample Input:
5 3 4 8 6 7

Sample Output:
2


Sample Input:
1 1 1

Sample Output:
-1



Memory limit: 256 MB
Time limit: 15 seconds
