So far, you've learned the basic arithmetic operators: addition (+), subtraction (-), multiplication (*), division (/), and integer division (//). Python also provides two additional operators:

  • exponentiation **

  • modulo (remainder) %

In this topic, we'll explore how these two operators work and when to use them.

Exponentiation (**)

The exponentiation operator ** is used to raise a number (the base) to the power of another number (the exponent).

print(2 ** 3)   # 8
print(5 ** 0)   # 1
print(10 ** 2)  # 100

This operator is useful when working with squares, cubes, and other powers:

print(5 ** 2)   # 25
print(2 ** 3)   # 8
print(4 ** 3)   # 64

You can often see powers in geometry and physics formulas. For example:

side = 4
print(side ** 2)   # area of a square: 16
print(side ** 3)   # volume of a cube: 64

Exponentiation grows numbers very quickly, so be careful when using very large values.

Exponentiation also has a high priority level in expressions. Consider the example:

print(-2 ** 2)    # -4
print((-2) ** 2)  # 4

Python applies exponentiation before unary minus. So -2 ** 2 is interpreted as -(2 ** 2).

Modulo (%)

The modulo operator % returns the remainder of dividing one number by another.

print(10 % 3)   # 1
print(14 % 5)   # 4
print(21 % 7)   # 0

A common use case is determining whether a number is even or odd:

print(7 % 2)   # 1, odd
print(8 % 2)   # 0, even

Some additional examples:

print(4 % 4)      # 0
print(55 % 77)    # 55
print(-10 % 3)    # 2
print(10 % -3)    # -2

Python ensures that the result of % always has the same sign as the divisor.
Taking the remainder with zero will result in:

ZeroDivisionError: integer division or modulo by zero
A quick note on negative numbers

Both operators behave a little differently when negative numbers are involved.

Exponentiation

print(-2 ** 2)  # Output: -4 (not 4!)
print((-2) ** 2) # Output: 4

Why? Because Python applies exponentiation before negation. So \(-2 ** 2\) is interpreted as \(-(2 ** 2)\).

Modulo

print(-10 % 3)  # Output: 2
print(10 % -3)  # Output: -2

Python ensures the result of % always has the same sign as the divisor.

Operation priority

Here is the updated priority list including the new operations:

  1. exponentiation **

  2. unary minus

  3. multiplication, division, integer division, remainder

  4. addition and subtraction

If multiple operations have the same priority, Python evaluates them from left to right.

print(2 ** 3 * 2)  # 16

To change the order explicitly, use parentheses:

print((2 ** 3) * 2)  # 16
Conclusion

In this lesson, you’ve learned:

  • ** lets you raise numbers to a power.

  • % helps you find the remainder in division.

  • Python's operator precedence and sign behavior can impact your results.
