Write a program that takes a string as input and returns a new string where all the characters within double quotation marks are replaced by their ASCII values. If a character inside the quotation marks is already represented as an ASCII value, leave it as it is.
Sample Input:
"Hello, World!"

Sample Output:
"72, 101, 108, 108, 111, 44, 32, 87, 111, 114, 108, 100, 33"


Sample Input:
"12345"

Sample Output:
"49, 50, 51, 52, 53"



Memory limit: 256 MB
Time limit: 15 seconds
